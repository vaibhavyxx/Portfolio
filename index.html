Game of Life

I developed a Game of Life simulation using C# for an academic project. The Game of Life is a cellular automaton devised by the mathematician John Conway, where cells evolve based on predefined rules. My implementation utilized object-oriented programming principles and advanced data structures to create a dynamic and interactive simulation.

The Game of Life project showcases several key features aimed at enhancing user experience and functionality. It includes a Random Board Generation feature, allowing for the creation of diverse initial board configurations with unique starting patterns for each simulation run. The Display Board functionality provides users with a visual representation of the evolving cellular automaton, enabling them to observe the state of the game board in real-time. Additionally, File I/O Operations have been implemented to facilitate loading and saving game configurations from external files, empowering users to specify custom starting patterns and save their progress for future sessions. The project also incorporates Advanced Simulation Logic, employing complex algorithms and 2D array manipulation techniques to implement the core rules of the Game of Life, including cell survival, birth, and death. Furthermore, Extra Credit Features, such as calculating the percentage of live cells on the board and automatically resetting the board under certain conditions, have been included to enhance the overall functionality and user experience of the application.

This project challenged me to apply theoretical programming concepts to a real-world problem, enhancing my understanding of data structures, algorithms, and software design principles. It provided valuable hands-on experience developing complex simulations and reinforced my passion for creating innovative solutions through code.
